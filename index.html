使用说明
  1. 输入 API Key。
  2. 上传模特图和服装图。
  3. 可以直接点“立即生成当前任务”。
  4. 也可以反复改图、改提示词，每次点“加入队列”。
  5. 最后点一次“开始队列”，系统会自动一个接一个跑完。
